Javascript object and its Representations

In JavaScript, objects are king. If you understand objects, you understand JavaScript.


Objects are Variables:

  1. JavaScript variables can also contain many values.
  2. Objects are variables too. But objects can contain many values.
  3. Object values are written as name : value pairs (name and value separated by a colon).
  4. A JavaScript object is a collection of named values

Objects written as name value pairs are similar to:

  1. Associative arrays in PHP
  2. Dictionaries in Python

Object Methods:

  1. Methods are actions that can be performed on objects.
  2. Object properties can be both primitive values, other objects, and functions.
  3. An object method is an object property containing a function definition.
  4. JavaScript objects are containers for named values, called properties and methods.

Creating a JavaScript Object:

  1. With JavaScript, you can define and create your own objects.

There are different ways to create new objects:

  1. Create a single object, using an object literal.
  2. Create a single object, with the keyword new
